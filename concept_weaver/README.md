# Concept Weaver

Concept Weaver is a cyberpunk-styled, interactive web application that deconstructs topics into knowledge graphs using a local LLM via Ollama.

It provides an agentic workflow where you input a concept, and the system autonomously queries an LLM to generate a structured graph of related sub-topics, historical context, and applications, which is then rendered as an interactive, physics-based network diagram.

## Features
- **Zero-Dependency Backend**: Runs on Python's built-in `http.server`.
- **Local AI Powered**: Seamlessly integrates with a local [Ollama](https://ollama.com/) instance to generate knowledge graphs structure on the fly. Auto-detects available models (prioritizing `llama3`).
- **Cyberpunk UI**: Features a modern, glowing dark mode interface with pulsing animations.
- **Interactive Graphs**: Uses `vis-network` to render draggable, physics-based nodes.
- **Detailed Context**: Click on any node to slide out a panel revealing detailed information generated by the LLM.

## Prerequisites
- Python 3.x
- [Ollama](https://ollama.com/) running locally on the default port `11434`.
- Ensure you have pulled at least one model, e.g., `ollama pull llama3`.

## How to Run
1. Navigate to the `concept_weaver` directory:
   ```bash
   cd concept_weaver
   ```
2. Start the server:
   ```bash
   python app.py
   ```
3. Open your browser and navigate to `http://localhost:8000`.
4. Enter a topic in the search bar (e.g., "Quantum Computing", "The Roman Empire", "Neural Networks") and click "Weave".

## Note
The application requests the LLM to output pure JSON. The success of the graph generation depends on the model's ability to strictly adhere to the requested JSON format. Models like `llama3` perform exceptionally well at this task.
